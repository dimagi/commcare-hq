Import a new build from the build server

  1. Browse builds
  2. Click on a CommCare version (example: "CommCare 2.13", NOT "CommCare ODK 2.13")
  3. Click on "View" in the Artifacts column for the build you want
  4. Right-click on "Download all (.zip)" and select "Copy Link Address"
  5. Paste the address into the box below
  6. Click Submit
  7. Wait until the page loads. If successful click back and refresh to see your new build.
