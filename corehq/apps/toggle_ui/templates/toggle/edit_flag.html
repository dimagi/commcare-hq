This is a utility to allow us to release or remove feature flags without a code deploy, and with a much easier way to walk back the change. This makes the code changes a lower risk cleanup step that won't affect any users.

The default state for a feature flag is "active" - where it can be turned on or off for individual users or domains. If we are to remove a feature flag, you can first set it to "disabled" here, which will behave the same as removing everything from the list. Likewise to make a feature flag generally available (GA), set it to "released," and it will behave as if you've enabled the flag for all projects and users.
