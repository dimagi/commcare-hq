Best Practices

In an ideal world, you should try your best to follow the suggestions / rules outlined here.

  • All forms for a Django App should be either in a forms.py file at the root of the app or a forms module.
  • Form class names should always end with Form.
  • Your imports should generally look like: 
    from django import forms
from django.utils.translation import ugettext_lazy, ugettext as _
from crispy_forms.helper import FormHelper
from crispy_forms import layout as crispy
from crispy_forms import bootstrap as twbscrispy
from corehq.apps.style import crispy as hqcrispy
