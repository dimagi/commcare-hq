Short titles: one line ok

However, longer titles (like a sentence) should be indented.

This is paragraph text within a block element. It should always follow standard indentation nested within its parent element. If you have a link within the paragraph, that's fine to leave inline with text, as well as code tags or span tags, etc. Long links, even though they are inline elements, can also behave more like block elements within the paragraph, to reduce the overall line length of that text.

Short paragraph text should also be on its own line within a p parent.
