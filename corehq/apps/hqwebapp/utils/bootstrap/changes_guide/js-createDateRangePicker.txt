The date range picker plugin can now be replaced with the B5 date/time picker.

Our new B5 date/time picker has support for date ranges. We no longer need a separate
widget for date ranges, so we can support a more consistent user experience.

See the style guide for examples tailored to CommCareHQ:
https://www.commcarehq.org/styleguide/b5/molecules/dates-times/

Old docs: https://github.com/dangrossman/daterangepicker
New docs: https://getdatepicker.com/
